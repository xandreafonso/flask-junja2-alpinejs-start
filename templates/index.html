Arquitetura dos Templates

1

Template Base (template.html)

Contém a estrutura HTML raiz, meta tags, CSS e JavaScript. Define blocks básicos para herança.

2

Layout Templates (layouts/)

Herdam do template base e definem a estrutura da página (header, nav, main, footer). Criam blocks específicos.

3

Page Templates (pages/)

Herdam dos layouts e contêm apenas o conteúdo específico de cada página/funcionalidade.
